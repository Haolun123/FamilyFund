# DESIGN: 定投策略回测（Backtest）

**状态**: ✅ 已实现（2026-04-21）
**日期**: 2026-04-17

---

## 一、目标

回答核心问题：**PE×VIX/QVIX 矩阵策略，相比固定金额定投，是否真的更优？**

通过历史数据模拟两种定投策略，对比最终收益、风险指标，验证市场温度计矩阵的有效性。

---

## 二、策略定义

| 策略 | 说明 |
|------|------|
| **固定金额策略** | 每期投入固定基准金额（如 ¥1000），无论市场状态 |
| **PE×VIX 矩阵策略** | 每期投入 = 基准金额 × 当期矩阵倍数，低估多投、高估少投或暂停 |

两种策略使用**相同的基准金额和回测区间**，确保可比性。

## 四、计算逻辑

### 每期执行流程

```
for each period (月/周):
    1. 取当期指数收盘价
    2. 取当期 PE 值
    3. 取当期 VIX / QVIX 值
    4. 查矩阵 → 倍数（暂停=0x, 0.3x ~ 顶格；"观望"概念已废弃，统一视为暂停=0x）
    5. 固定策略：买入金额 = base_amount
       矩阵策略：买入金额 = base_amount × 倍数
    6. 买入份额 = 买入金额 / 当期价格
    7. 累计份额 += 新份额；累计成本 += 买入金额

期末汇总:
    终值 = 累计份额 × 最新价格
    XIRR = scipy.optimize.brentq(npv_func)
    最大回撤 = 复用 _compute_max_drawdown_series()
```

### "顶格"的处理

矩阵中的"顶格"映射到具体倍数：权益类顶格 = 10x，黄金顶格 = 5x（均为默认值，UI 可配置）。

### "观望"概念（已废弃）

已废弃"观望"分级，统一归并为"暂停"（0x）。矩阵中所有非数字倍数格（原"观望"/"暂停"）均不投入。

## 六、核心技术难点

### 历史 PE 数据（最大风险点）

yfinance 的 `trailingPE` 仅返回**当前值**，无历史序列。美股历史 PE 方案调研结论：

| 方案 | 可行性 | 说明 |
|------|------|------|
| **A. multpl.com 爬取** | 中 | S&P500 月度 PE 有公开数据，但需要爬虫，稳定性差 |
| **B. FRED API** | 低 | FRED 无独立 PE 序列，仅有价格，PE 需另外推算 |
| **C. Shiller Yale Excel** ✅ | 高 | Yale 公开数据，月频，1871 年至今，直接下载，无需 API Key |
| **D. 简化：只回测 A 股** | 高（最简单）| 先实现 A 股回测，美股待 PE 数据源确认后补充 |
| **E. 用固定 PE 阈值区间模拟** | 低 | 失真太大，不建议 |

**已确认方案**：
- **标普500**：采用方案 C，Shiller Yale Excel 提供月频 Trailing PE，直接 `pd.read_excel(url)` 读取
- **纳指100**：⚠️ 无干净免费数据源（macrotrends.net 需 JS 渲染，Quandl 无 NDX 系列）→ **用标普 PE 作代理**

### 纳指100 PE 代理局限说明

纳指100 回测使用标普500 PE 作为 PE 信号，存在以下偏差：

| 局限 | 影响 |
|------|------|
| 纳指 PE 历史上高于标普（科技溢价）| 用标普 PE 判断时，纳指实际估值更贵，矩阵会系统性低估风险 |
| 两者相关性约 0.85，方向一致但绝对值不同 | 趋势方向准确，但分档阈值（如 PE>32 暂停）对纳指偏宽松 |
| 在科技泡沫期（2000年、2021年）偏差最大 | 纳指 PE 可高达 80-100x，标普同期仅 30-40x |

**在 Dashboard 和回测报告中均需显示此局限说明**。

## 十七、策略的哲学基础

### 为什么矩阵策略只用于指数，不用于个股

矩阵策略的核心前提是**市场层面的均值回归**：市场整体估值会高会低，但不会归零。

指数基金天然具备反脆弱机制——成分股会被定期替换，表现差的踢出，表现好的纳入。沪深300、标普500、纳指100都在做这件事。买指数，你买的不是某家公司，而是这个自我更新的规则本身。

因此 PE×VIX 矩阵在这里是成立的：**市场现在贵不贵、怕不怕**，有均值回归作为锚。

个股没有这个锚。PE 低可能是价值洼地，也可能是价值陷阱——公司真的可以经营不善、破产消亡。对个股的决策应该问的是"这家公司好不好"，这不是均值回归能回答的问题，这是第十人系统的用武之地。

> **矩阵策略是在问"市场现在贵不贵、怕不怕"，而不是在问"这家公司好不好"。前者有均值回归作为锚，后者没有。**

### 关于历史数据的正确使用姿势

回测不是预测机器，而是**验证策略逻辑是否内部自洽**的工具。

PE×VIX 矩阵的逻辑——"低估+恐慌时多买"——不是从历史数据归纳出来的，而是有基本面依据的先验判断。回测只是在问：如果这个逻辑是对的，它在数字上体现出来了吗？

回测的价值不是"预测未来收益"，而是：
- 给了你**不执行某个策略的理由**（黄金矩阵与其驱动力不匹配）
- 给了你**继续执行的信心**（权益类指数的均值回归被10年数据验证）
- 识别了**策略的边界条件**（矩阵对趋势型行情失效，如黄金2024）

巴菲特鄙视的不是历史数据，而是把历史价格波动率当作风险的做法。他自己也大量使用历史数据——看企业历史盈利能力和护城河的持续性。本质上都是在问：过去证明是稳定的东西，未来有没有理由继续稳定？

## 十六、实证回测结论（2026-05）

基于四象限散点图的实际回测结果。

### 全标的统一 2015 年起（10年样本，最可靠结论）

QVIX 数据源升级后（50ETF，2015-02起），A股回测可从2015年起，与标普/纳指/黄金统一起始，构成迄今最完整的对比。10年样本覆盖2015股灾、2018熊市、2020-2021牛市、2022-2024震荡。

**核心发现：所有标的矩阵策略 XIRR 均有正超额（0.4-0.8%），全部在第一象限右侧。**

| 标的 | 象限 | XIRR超额 | 绝对盈亏超额 | 解读 |
|------|------|---------|------------|------|
| 纳指100 | I 多投多赚 | +0.8% | +400k | 最强，双赢，矩阵效果最显著 |
| 沪深300 | I 多投多赚 | +0.65% | +150k | 稳定有效，均值回归特征充分验证 |
| 中证A500 | I 多投多赚 | +0.5% | +200k | 与沪深300类似，绝对盈亏更高 |
| 标普500 | I 多投多赚 | +0.4% | +100k | 温和稳定，危机加仓价值体现 |
| 黄金（原始）| I/边界 | +0.5% | ≈0 | XIRR有超额但绝对平手，2024异常值被稀释 |
| 黄金（对冲）| IV 多投无超额 | +0.5% | -200k | 明确失效，对冲逻辑在10年维度不成立 |

**修正说明：**
- 黄金原始矩阵比此前（5年样本）结论更好——XIRR 有超额，10年视角下2024单边行情影响被稀释。但绝对盈亏仍接近零，性价比低于其他标的，固定定投仍是更简洁的选择。
- 黄金对冲矩阵在10年维度下确认失效（第四象限），维持"固定定投"建议。
- 此前"标普长期踏空"的结论已完全推翻，10年视角下标普矩阵策略温和有效。

## 十四、回测指标解读指南

### 核心原则

回测只回答一个问题：**PE×VIX 矩阵策略，在选时上是否有效？**
不回答标的选择对不对，也不回答绝对收益是否满意。

### 主要对比指标

| 指标 | 含义 | 局限 |
|------|------|------|
| **XIRR** | 资金加权年化收益率，考虑每笔投入的时间和金额 | 忽略规模，投入小的标的 XIRR 高也不代表赚了更多 |
| **绝对盈亏** | 实际赚了多少钱（CNY） | 与投入规模挂钩，不反映效率 |
| **每元成本市值** | `final_value / total_cost`，投入1元最终拿回多少 | 不考虑时间，长周期天然更高 |
| **最大回撤** | 策略在历史上的最大亏损幅度 | 反映风险，矩阵策略应优于固定策略 |

### 如何读懂固定 vs 矩阵的对比

两种策略总投入**可能不同**（矩阵在高估时少投、低估时多投），这是正常的，不影响对比有效性——XIRR 已经消化了这个差异。

对角线分析框架：

| 场景 | 矩阵总投入 | 矩阵绝对盈亏 | 结论 |
|------|----------|------------|------|
| 多投多赚 | > 固定 | > 固定 | 低估时加仓有效，策略跑赢 |
| 少投少亏 | < 固定 | < 固定但亏损更小 | 高估时减仓有效，风险控制好 |
| 多投盈亏相近 | > 固定 | ≈ 固定 | 加仓未带来超额，策略无效 |
| 少投XIRR更高 | < 固定 | < 固定 | 选时准确但仓位保守，效率高但绝对贡献小 |

### 回测模型假设

- **资金假设**：每期按倍数投入的资金均能到位，由用户 Cash 储备（约10万）保证。系统不模拟资金约束。只要 Cash 储备充足，模型不会失效。
- **不做虚拟货币基金**：矩阵少投的差额"去哪了"是用户个人决策（消费/存款/其他投资），系统无法模拟，强行假设会制造假精确。
- **对比标准**：XIRR 是天然公平的对比指标（考虑了每笔现金流的时间和金额），不需要等资金约束。

### 回测回答不了的问题

- **标的选择对不对**：看 Tab1 基准对比（你的基金 vs 沪深300/标普）
- **某笔具体调仓对不对**：看 P3 调仓决策质量复盘（待 transaction.csv 积累后实现）
- **绝对收益是否足够**：看 Tab1 财务独立测算（FI 进度）

## 十五、矩阵策略的适用边界

### 核心前提

**矩阵策略是周期性工具，核心价值在危机时的逆向加仓。**

有效前提 = 市场最终会回来（指数的反脆弱机制保证），且投资者有足够耐心和弹药。

### 各标的适用性（基于最终验证结论）

| 标的 | 回测期 | 矩阵效果 | 核心发现 |
|------|--------|---------|---------|
| 纳指100 | 24年 | ✅ 最强 | 超额+367万，每元效率最高。科技泡沫破裂+2022调整提供极佳买点 |
| 标普500 | 24年 | ✅ 有效 | 超额+143万。**早期"长期慢牛踏空"结论是基于5年短数据的误判**，20年+视角下矩阵有效，核心价值在危机时顶格加仓（2001/2008/2020），而非日常高PE减仓 |
| 沪深300/A500 | 10年 | ✅ 有效 | 均值回归明确，超额稳定。但数据期较短，结论可信度低于美股 |
| 日经225 | 24年 | ✅ 有效 | 超额+153万，每元效率5.01x，略高于标普。但需要忍受11年不赚钱的心理压力 |
| DAX | 24年 | ✅ 有效但弱 | 超额+84万，欧洲增长慢导致绝对收益最低 |
| KOSPI | 24年 | ⚠️ 质量差 | 超额靠投入量大而非选时准确，每元效率反而低于固定定投 |
| 黄金 | 10年 | ⚠️ 固定更优 | 两套矩阵（原始+对冲）均在结构性牛市中失效。黄金驱动因素（去美元化/央行购金）与PE/VIX/乖离率信号无关 |

### 关于"标普长期慢牛"的修正

早期分析（基于2020-2024的5年数据）得出"标普慢牛，矩阵系统性踏空"的结论。这是**错误的**。

修正过程：
1. 5年数据（2020-2024）：只含疫情后牛市，矩阵在高PE时减仓确实踏空 → 错误结论
2. 10年数据（2015-2025）：包含2022加息熊市，矩阵温和跑赢 → 初步修正
3. 20年数据（2005-2025）：包含2008金融危机，矩阵+150万超额 → 验证有效
4. 24年数据（2001-2025）：包含科技泡沫后+金融危机+疫情，矩阵+143万超额 → 最终结论

**根本原因**：标普确实是"长期慢牛"，但每隔数年会经历一次-30%~-50%的深度回调（2001/2008/2020/2022），矩阵策略的价值正是在这些回调中顶格加仓积累低价份额。"慢牛"和"矩阵有效"并不矛盾——矩阵不需要PE"回归均值"，只需要周期性恐慌给出加仓机会。

### 黄金为什么例外

黄金没有内在盈利，不受PE/VIX逻辑约束。2024年的上涨由去美元化、央行购金、地缘政治驱动——这些因素与市场恐慌无关，无法被任何基于波动率或估值的矩阵捕捉。黄金的结论维持：**固定定投，不择时**。

### 对实际操作的最终建议

| 标的 | 建议 |
|------|------|
| 纳指100 / 标普500 / 沪深300 / A500 | 继续矩阵信号，正常执行 |
| 黄金 | 固定定投，矩阵信号仅供参考 |
| 日经225（如未来配置）| 可复用标普PE×VIX信号 |
| KOSPI（不推荐）| 信号选时质量差 |

## 十八、红利低波 ETF（512890）利差策略回测（2026-05）

### 研究背景

红利类资产的估值锚不是 PE×VIX，而是**股息率与无风险利率的利差**。reference/IMG_1177.JPG 展示了"红利双轨"三因子框架，核心逻辑是：股息率 > 国债+1pp 时才具备定投性价比。

本次回测验证：对 512890（红利低波 ETF）用利差信号择时，是否优于固定定投。

### 数据来源

| 数据 | 来源 | 区间 |
|------|------|------|
| 512890 价格 | yfinance `512890.SS`（前复权）| 2019-01 ~ 2026-05 |
| 股息率（近似）| 上证红利指数滚动PE倒数 × 40%分红比例 | 2019+ |
| 中国10Y国债 | akshare `bond_zh_us_rate` | 2019+ |

> 注：用 `1/PE × 40%` 近似实际股息率，误差约±1pp，方向完全一致。

### 回测结果

四种阈值（0/0.5/1/1.5pp）结果完全相同——**利差策略在此期间退化为固定定投**，XIRR 均为 10.4%，零暂停期。

### 原因分析

历史利差分布（2019-2026，股息率近似值）：

| 分位数 | 利差 |
|--------|------|
| 10th   | 2.29pp |
| 50th（中位）| 3.75pp |
| 90th   | 4.93pp |
| 当前（2026-05）| 2.76pp |

**在2019-2026年间，利差最低也有1.90pp，从未低于1pp阈值。** 根本原因是国债10Y长期维持在1.6-1.9%的低位，红利股相对国债永远处于"便宜"区间，阈值策略没有触发机会。

### 结论

1. **512890 当前使用固定定投即可**，不需要利差择时策略
2. **利差策略的适用前提**：国债收益率回升到3-4%（如2013-2018年水平），利差才会有分化，策略才有意义
3. **"红利双轨"框架本身合理**，但在低利率环境下股息溢价条件常态化满足，实际只有"PE分位"和"ROE稳定性"还有区分度
4. **红利类资产更适合固定定投**，让股息自然复利；矩阵择时对均值回归特征更强的宽基指数更有效

## 十九、全球市场回测验证：标普PE×VIX信号的跨市场有效性（2026-05）

### 研究背景

矩阵策略使用标普500 PE×VIX作为信号。问题：这套信号能否代理全球风险，应用于非美市场的定投择时？

### 相关性验证（2001-2026，300个月）

月度收益率相关矩阵：

| | 标普500 | 日经225 | KOSPI | DAX |
|--|--------|--------|-------|-----|
| 标普500 | 1.00 | 0.65 | 0.64 | 0.81 |
| 日经225 | 0.65 | 1.00 | 0.62 | 0.62 |
| KOSPI | 0.64 | 0.62 | 1.00 | 0.65 |
| DAX | 0.81 | 0.62 | 0.65 | 1.00 |

**极端行情同步性（标普跌>5%的33个月）：**
- 日经同步下跌率 97%，平均跌 -7.8%
- KOSPI 同步下跌率 91%，平均跌 -7.7%
- DAX 同步下跌率 97%，平均跌 -9.3%

结论：危机时全球股市高度同步，美股PE×VIX作为"全球恐慌加仓"信号是可靠的。

### 回测结果（2001-06 ~ 2026-05，月频，base=¥1000，信号=标普PE×VIX）

| 市场 | 固定盈亏 | 矩阵盈亏 | 超额 | 固定每元市值 | 矩阵每元市值 |
|------|---------|---------|------|-----------|-----------|
| 纳指100 | +288万 | +655万 | +367万 | 10.61 | 11.62 |
| 日经225 | +95万 | +248万 | +153万 | 4.15 | 5.01 |
| 标普500 | +101万 | +244万 | +143万 | 4.35 | 4.95 |
| KOSPI | +114万 | +207万 | +93万 | 4.80 | 4.35 |
| DAX | +68万 | +152万 | +84万 | 3.26 | 3.46 |

暂停期均为29期（PE高估+VIX低时暂停，各市场共用同一信号，暂停期相同）。

### 关键发现

1. **纳指100遥遥领先**：绝对收益是其他的3-4倍，矩阵超额+367万，科技泡沫破裂和2022调整提供极佳买点
2. **日经225 ≈ 标普500**：超额+153万略高于标普+143万，每元效率也更高（5.01 vs 4.95）。2001-2012长期底部被矩阵大量加仓，2013后日经从8000涨到40000
3. **KOSPI超额靠量不靠质**：每元市值矩阵（4.35）反而低于固定（4.80），说明PE×VIX信号对韩国市场选时不准确，超额完全来自投入量更大
4. **DAX最弱**：有效但绝对收益最低，欧洲增长慢，危机后涨幅有限

### 投资启示

- **日经225是最适合复用现有矩阵策略的非美标的**：危机同步率高（97%），长期底部积累提供极强回报
- **KOSPI不适合**：韩国市场有自己的节奏（半导体周期、朝鲜地缘），美股PE×VIX信号选时不准
- **DAX可用但弱**：效果最小，分散价值有限
- **现有信号无需修改**：不需要为非美市场开发新矩阵，标普PE×VIX直接可用
