# DESIGN: AI 周度评估（AI Weekly）

> **状态**：✅ 已实现（`src/ai_weekly.py` + Dashboard Tab1）
> **日期**：2026-04-25（实现 2026-04-30）

---

## 功能定位

每次 Weekly Update 完成后，基于 FamilyFund 自有数据由 GLM 生成 3-5 句中文周报，展示在 Dashboard。与同花顺的差异：结合**具体持仓权重和温度计信号**，给出有操作指向的分析，而非通用市场评述。

---

## 架构

- **模型**：`glm-4-flash`（免费，文本生成模型）
- **API**：智谱 GLM，公司设备直连可用（`open.bigmodel.cn`）
- **费用**：免费额度内，约 ¥0/次
- **触发**：手动点击「生成本周 AI 点评」按钮
- **配置**：`$FAMILYFUND_DATA/tenth_man_config.json`（与第十人系统共用）

---

## 数据预组装（Python 侧）

`src/ai_weekly.py` → `build_weekly_context()` 负责组装，GLM 只收到文字摘要：

1. **基金总览**：NAV、总资产、本周净值变化、年化收益、最大回撤、XIRR、夏普
2. **资产配置**：各类别市值和权重（排除 Cash）
3. **本周各类别净值表现**：每个类别本周涨跌幅
4. **市场温度计信号**：VIX、QVIX、美债收益率、五标的定投倍数建议
5. **再平衡提示**：偏差 ≥5% 的类别自动列出

---

## System Prompt

```
你是一位家庭基金的投资顾问助理。基于以下数据，用简洁的中文写一段3-5句的周报。
要求：
1. 点评本周净值表现的主要驱动因素（哪个类别贡献最大）
2. 结合温度计信号，给出下周定投方向建议
3. 如有明显超配/低配，提醒关注再平衡
4. 语气专业简洁，有具体数字支撑，不废话
5. 直接输出周报正文，不需要标题
```
