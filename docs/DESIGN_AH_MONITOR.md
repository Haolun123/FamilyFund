# DESIGN: AH 股溢价监测（AH Premium Monitor）

> **状态**：已实现
> **日期**：2026-05-07

---

## 功能定位

同一家公司在 A 股和港股同时上市，同股不同价。溢价率衡量 A 股相对港股的贵贱程度，为价值投资者提供"哪个市场买入更有安全边际"的参考。

**不做套利信号**：价差长期存在（资金流通限制、税差、流动性），监测的目的是价值投资选市场，不是快速套利。

---

## 溢价率计算

```
溢价率 = A股价格 / (H股价格 × HKD/CNY汇率) × 100
```

| 溢价率 | 信号 | 含义 |
|--------|------|------|
| > 120 | 🟢 港股便宜 | A股溢价超20%，港股有明显安全边际 |
| 90–120 | 🟡 接近平价 | 两地差异不大，流动性折价可解释 |
| < 90 | 🔴 港股贵 | 少见，港股比A股贵（如招商银行历史上出现过）|

历史均值约 130（A股长期有溢价），但个股差异很大，需参考个股1年历史分位数。

---

## 注意事项

使用溢价率时需叠加考虑：

| 因素 | 说明 |
|------|------|
| 股息税 | 港股通持有H股，股息征10%预提税；A股免税。对高股息股影响显著 |
| 流动性折价 | H股流动性普遍低于A股，部分折价是合理补偿（尤其区域性银行）|
| 汇率风险 | H股以港币计价，人民币升值侵蚀港股收益 |
| 科创板溢价 | 中芯国际等科创板标的A股溢价极高（历史200+），主要是散户情绪 + 流动性溢价，非价值信号 |

## 数据结构

### `$FAMILYFUND_DATA/ah_config.json`

```json
{
  "stocks": [
    {"name": "中海油",   "a_code": "600938.SS", "h_code": "0883.HK"},
    {"name": "招商银行", "a_code": "600036.SS", "h_code": "3968.HK"},
    {"name": "中芯国际", "a_code": "688981.SS", "h_code": "0981.HK"},
    {"name": "重庆银行", "a_code": "601963.SS", "h_code": "1963.HK"}
  ],
  "_cache": {
    "600938.SS": {"price": 38.28, "updated": "2026-05-07"},
    "0883.HK":   {"price": 27.28, "updated": "2026-05-07"}
  },
  "_history": {
    "600938.SS/0883.HK": [
      {"date": "2026-05-07", "premium": 151.9}
    ]
  }
}
```

- `_cache`：当日价格缓存，`updated == today` 则不重复拉取
- `_history`：近365天每日溢价率，用于计算1年历史分位数（积累10条以上才显示）

---

## 数据源

- **价格**：yfinance `fast_info.last_price`（A股 `.SS`/`.SZ`，港股 `.HK`）
- **汇率**：复用 `fx_service.get_rate('HKD', 'CNY')`，失败时 fallback 0.924

akshare `stock_zh_ah_spot_em()` 在公司网络下超时，已放弃。

---

## 设计决策

- **动态增删标的**：不 hardcode，用户在 UI 内管理，存 `ah_config.json`
- **历史分位数**：`_history` 内嵌在配置文件，每日自动追加，保留近365天，无额外数据库依赖
- **Tab5 臃肿问题**：个股基本面、AH 溢价、DCA 均改为 `expander` 默认收起，按需展开
