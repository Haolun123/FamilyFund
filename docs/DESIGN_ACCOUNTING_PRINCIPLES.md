# FamilyFund 家庭记账原则

> **版本**: v1.0  
> **日期**: 2026-04-27  
> **适用范围**: 鲨鱼记账 App + balance_sheet.csv + cashflow_log.csv 的协同使用规范

---

## 核心原则：资产负债表驱动

记账的目的不是记"花了多少钱"，而是记录**净资产的变化**。

每一笔交易都可以用以下框架分析：

```
△资产 = △负债 + △净资产（收益/损失）
```

**判断口诀：**

| 场景 | 本质 | 如何记录 |
|------|------|---------|
| 钱花出去，什么都没留下 | 净资产减少 | ✅ 鲨鱼记账：支出 |
| 钱出去，换回了资产 | 资产形态转换 | ❌ 不记支出，更新 balance_sheet 对应行 |
| 钱进来，来自外部（工资/补贴/租金） | 净资产增加 | ✅ 鲨鱼记账：收入 |
| 钱进来，来自借贷 | 负债增加，净资产≈不变 | ❌ 不记收入，更新 balance_sheet 负债行 |
| 负债还款（本金部分） | 负债减少，净资产≈不变 | ❌ 不记支出，更新 balance_sheet 贷款余额 |
| 负债还款（利息/手续费部分） | 净资产减少 | ✅ 鲨鱼记账：支出「贷款利息」或「贷款手续费」；免息贷款整个生命周期不记 |
| 资产估值变化（不动产涨跌/基金浮亏等） | 账面价值变化，无现金流 | ❌ 不记鲨鱼记账，每季末更新 balance_sheet 对应行估值；季度财报瀑布图自动体现 |

## 三层数据分工

| 工具 | 记录内容 | 频率 | 用途 |
|------|---------|------|------|
| **鲨鱼记账** | 影响净资产的日常流量（收入/支出） | 每笔实时记录 | 季度消费汇总、Sankey 流图数据源 |
| **balance_sheet.csv** | 所有资产/负债的季末存量快照 | 每季末更新 | 家庭净资产计算、QoQ 对比 |
| **cashflow_log.csv** | 基金外特殊现金流（不经过鲨鱼记账的项目） | 按需，每季末检查 | 季度财报现金流补充 |

## 复杂案例：换车事件完整记账

> 场景（实际情况）：原持有宝马 X5（balance_sheet 提计 ¥30万）。将 X5 赠予父亲，
> 取得父亲的 X3 并卖出变现 ¥20.7万，同时领取政府以旧换新补贴 ¥1.5万。
> 贷款 ¥30万 + 首付 ¥1.9万，购入蔚来 ES8（提计 ¥30万）。
>
> **净资产实质变化：-¥30万（X5赠予）+ ¥20.7万（X3变现）+ ¥1.5万（补贴）- ¥1.9万（首付现金支出）= -¥9.7万**

### 逐笔拆解

**① X5 赠予父亲（¥30万资产流出）**

- 本质：无偿赠予，净资产减少 ¥30万
- 鲨鱼记账：✅ 记支出，类别「长辈」，金额 ¥30万，备注「X5赠予父亲」
- balance_sheet：下季末删除 X5 对应行（`Asset_RealEstate`）

> **注**：赠予金额填 balance_sheet 上的提计残值（¥30万），而非市场价，
> 保持与资产负债表口径一致。

**② X3 卖出变现 ¥20.7万到账**

- 本质：X3 是父亲的资产，变现后转给你作为换车补偿，视为收入
- 鲨鱼记账：✅ 记收入，类别「其他收入」，金额 ¥20.7万，备注「X3卖出（父亲补偿换车）」
- balance_sheet：无需操作（现金余额自动体现）

**③ 政府以旧换新补贴 ¥1.5万到账**

- 本质：外部转入，净资产增加
- 鲨鱼记账：✅ 记收入，类别「其他收入」，金额 ¥1.5万，备注「以旧换新政府补贴」
- balance_sheet：无需操作

**④ 贷款 ¥30万放款**

- 本质：负债增加，净资产≈不变
- 鲨鱼记账：❌ 不记
- balance_sheet：`Liability_LongTerm` 新增车贷行 ¥30万

**⑤ 首付 ¥1.9万 + 购车款（含贷款）→ 蔚来 ES8**

- 本质：现金 + 贷款 → 车辆（资产形态转换）
- 鲨鱼记账：❌ 不记（首付是资产转换，不是消耗）
- balance_sheet：`Asset_RealEstate` 新增蔚来 ES8 行，提计 ¥30万

**⑥ 每月还款 ¥5,000（免息车贷）+ 电池租赁 ¥1,068**

- ¥5,000 本金还款：负债减少、现金减少，净资产不变
  - 鲨鱼记账：❌ 不记
  - balance_sheet：每季末查银行 App，更新 `Liability_LongTerm` 车贷剩余本金
- ¥1,068 电池租赁（BaaS 服务费）：纯消费，净资产减少
  - 鲨鱼记账：✅ 记支出，类别「交通」，备注「蔚来电池租赁」

> **有息贷款的通用原则**：鲨鱼记账只记**实际支付的利息和手续费**（净资产减少部分），
> 本金还款不记。若为一次性手续费（如部分"免息"贷款收服务费），在支付时记支出「贷款手续费」。

### 换车事件汇总

| 项目 | 鲨鱼记账 | balance_sheet 变化 |
|------|---------|-------------------|
| X5 赠予父亲 | ✅ 支出 ¥30万「长辈」 | Asset_RealEstate 删除 X5 行 |
| X3 卖出 ¥20.7万 | ✅ 收入 ¥20.7万「其他收入」 | 无（X3 不在你的 balance_sheet） |
| 政府补贴 ¥1.5万 | ✅ 收入 ¥1.5万「其他收入」 | 无 |
| 贷款 ¥30万 | ❌ 不记 | Liability_LongTerm +¥30万 |
| 首付 ¥1.9万 | ❌ 不记（资产转换） | Asset_RealEstate 新增蔚来 ¥30万 |
| 月供 ¥5,000（免息本金） | ❌ 不记 | 每季末更新贷款余额 |
| 电池租赁 ¥1,068/月（BaaS） | ✅ 支出「交通」 | 无 |

## 工资与收入的记录口径

### 记录原则

鲨鱼记账的收入 = **税后实际到账金额**，不区分谁的工资，统一记入家庭账本。

| 收入类型 | 鲨鱼记账 | 备注 |
|---------|---------|------|
| 工资（税后到账） | ✅ 记，类别「工资」 | 含奖金，以实际到账为准 |
| SAP ESPP/RSU 归属 | ✅ 如有现金到账则记 | 股票形式归属不记（已在 portfolio.csv 追踪） |
| 公积金（单位部分） | ❌ 不记 | 不经过你的银行账户，季末查余额更新 balance_sheet |
| 公积金（个人部分扣除） | ❌ 不记 | 已在税前扣除，工资到账金额已是税后 |
| 公积金（实际提取到账） | ✅ 记收入 | 类别「公积金」 |
| 房租收入 | ✅ 记收入 | 类别「房租」 |
| 理财/基金分红现金到账 | ✅ 记收入 | 类别「投资收益」 |
| 家庭基金取出 | ❌ 不记 | 已在 portfolio.csv Cash NCF 里追踪 |

## cashflow_log.csv 的定位

`cashflow_log.csv` **只补录鲨鱼记账捕捉不到的特殊项**，大多数季度可以为空。

需要填的情况：
- 大额现金交易（没有走银行，鲨鱼记账没有记录）
- 跨境转账
- 家庭内部大额借款/还款（如对父母的借款）

**不需要填的情况：**
- 工资、房租、消费 → 已在鲨鱼记账
- 家庭基金注资/取出 → 已在 portfolio.csv Cash NCF
- 贷款本金变化 → 体现在 balance_sheet 负债余额

## 季末对账流程（约 15-20 分钟）

1. **鲨鱼记账导出**：导出当季明细 CSV，放入 `$FAMILYFUND_DATA/<季度>/鲨鱼记账明细.csv`
2. **更新 balance_sheet.csv**：查各账户余额，更新存量快照（详见 USER_MANUAL.md 第 6 节）
3. **检查 cashflow_log.csv**：有无遗漏的基金外特殊项
4. **核对公式**：
   ```
   期末净资产 ≈ 期初净资产 + 鲨鱼收入合计 - 鲨鱼支出合计 ± 资产估值变化
   ```
   误差在 ¥5,000 以内属正常（估值误差、未记小额等）

## Sankey 家庭资金流图（未来功能）

当 cashflow_log 和鲨鱼记账数据积累 2-3 个季度后，可以做：

```
[流入]                    [流出/沉淀]
工资收入       ──────────→ 餐饮消费
公积金提取     ──────────→ 交通
房租收入       ──────────→ 孩子相关
其他收入       ──────────→ 购物
               ──────────→ 家庭基金注资
               ──────────→ 净储蓄增加
```

**前置条件**：
- 鲨鱼记账完整记录（已满足）
- 工资口径统一（已明确）
- balance_sheet 至少 2 个季度（已有 25Q4 + 26Q1）
