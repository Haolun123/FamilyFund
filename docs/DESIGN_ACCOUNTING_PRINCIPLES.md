# FamilyFund 家庭记账原则

> **版本**: v1.1
> **日期**: 2026-04-27
> **适用范围**: 鲨鱼记账 App + balance_sheet.csv + cashflow_log.csv 的协同使用规范

---

## 核心原则：双轨制记账

家庭财务管理需要同时回答两个问题，单一视角都会产生盲区：

| 问题 | 视角 | 工具 |
|------|------|------|
| 我的净资产增加了多少？ | 净资产视角（P&L） | balance_sheet + 鲨鱼记账（清洗后） |
| 我这个月实际还剩多少可支配？ | 现金流视角（Cash Flow） | 鲨鱼记账（原始，含「债务还本」） |

**流动性幻觉的危险**：若月供本金不记鲨鱼，鲨鱼结余会虚高，误以为可支配收入更多。
例：月收入 ¥4万，消费 ¥1.5万，车贷本金 ¥5,000 —— 鲨鱼结余 ¥2.5万，但真实自由现金流只有 ¥2万。

**解法（Gemini 双轨制方案）**：
- **前端（鲨鱼记账）**：所有现金流出都记，本金还款单独用「债务还本」一级分类
- **后端（FamilyFund）**：解析 CSV 时剔除「债务还本」，净资产公式保持正确；同时保留原始数据计算自由现金流和 DSCR

## 鲨鱼记账分类规范

### 「债务还本」一级分类（新增）

**用途**：记录所有贷款本金还款。钱从账户流出，但净资产不变（负债等额减少）。

| 子类 | 示例 | 金额来源 |
|------|------|---------|
| 房贷本金 | 每月还款明细中的本金部分 | 查银行 App 还款明细 |
| 车贷本金 | 蔚来免息车贷 ¥5,000/月 | 固定金额 |
| 其他贷款本金 | 按需 | 查贷款合同/App |

> **注**：LPR 浮动利率房贷每年 1 月调整，本金/利息比例随之变化，需每月查银行 App 确认当月拆分金额。

### 正常支出分类中的贷款相关项

| 分类 | 内容 | 本质 |
|------|------|------|
| 「贷款利息」 | 房贷利息部分 | 净资产减少，真实消费 |
| 「贷款手续费」 | 一次性服务费等 | 净资产减少，真实消费 |
| 「交通」→蔚来电池租赁 | BaaS ¥1,068/月 | 净资产减少，真实消费 |

## 判断口诀

每一笔交易用以下框架判断如何记录：

```
△资产 = △负债 + △净资产（收益/损失）
```

| 场景 | 本质 | 鲨鱼记账 | balance_sheet |
|------|------|---------|--------------|
| 钱花出去，什么都没留下 | 净资产减少 | ✅ 正常支出分类 | 无需操作 |
| 钱出去，换回了资产 | 资产形态转换 | ❌ 不记 | 更新对应资产行 |
| 钱进来，来自外部（工资/补贴/租金） | 净资产增加 | ✅ 正常收入分类 | 无需操作 |
| 钱进来，来自借贷 | 负债增加，净资产≈不变 | ❌ 不记 | 更新负债行 |
| 负债还款（本金） | 负债减少，净资产≈不变 | ✅ 「债务还本」分类 | 每季末更新贷款余额 |
| 负债还款（利息/手续费） | 净资产减少 | ✅ 「贷款利息」等正常支出 | 无需操作 |
| 资产估值变化（房产涨跌/基金浮亏） | 账面变化，无现金流 | ❌ 不记 | 每季末更新估值 |

## 三层数据分工

| 工具 | 记录内容 | 频率 | 用途 |
|------|---------|------|------|
| **鲨鱼记账** | 所有现金流出入（含「债务还本」） | 每笔实时记录 | 现金流管理、消费分析、Sankey 数据源 |
| **balance_sheet.csv** | 所有资产/负债的季末存量快照 | 每季末更新 | 净资产计算、QoQ 对比 |
| **cashflow_log.csv** | 基金外特殊项（不经过鲨鱼记账） | 按需，每季末检查 | 季度财报现金流补充 |

## FamilyFund 后端清洗逻辑（待实现）

解析鲨鱼 CSV 时，将「债务还本」单独剥离：

```python
df = parse_shark_csv(path)  # UTF-16, tab分隔

# 自由现金流视角（现金管理用）
free_cashflow = df[df['收支类型'] == '收入']['金额'].sum() \
              - df[df['收支类型'] == '支出']['金额'].sum()

# 净资产视角（财报用，剔除本金还款）
net_income = df[df['收支类型'] == '收入']['金额'].sum() \
           - df[(df['收支类型'] == '支出') & (df['类别'] != '债务还本')]['金额'].sum()

# 刚性债务（债务还本合计）
debt_service = df[(df['收支类型'] == '支出') & (df['类别'] == '债务还本')]['金额'].sum()

# DSCR = 收入 / 刚性债务还本
dscr = income / debt_service if debt_service > 0 else None
```

净资产核对公式（后端）：
```
期末净资产 ≈ 期初净资产 + 鲨鱼收入 - 鲨鱼支出（剔除债务还本）± 资产估值变化
```

**季度财报 Tab 可展示**：
- 自由现金流（当季）
- 净储蓄率 = 净收入 / 总收入
- DSCR = 总收入 / 刚性债务还本（建议 > 1.5x 为安全）

## 复杂案例：换车事件完整记账

> 场景：原持有宝马 X5（balance_sheet 提计 ¥30万）。将 X5 赠予父亲，
> 取得父亲的 X3 并卖出变现 ¥20.7万，同时领取政府以旧换新补贴 ¥1.5万。
> 贷款 ¥30万（免息）+ 首付 ¥1.9万，购入蔚来 ES8（提计 ¥30万）。
>
> **净资产实质变化：-¥30万（X5赠予）+ ¥20.7万（X3变现）+ ¥1.5万（补贴）- ¥1.9万（首付）= -¥9.7万**

### 逐笔拆解

**① X5 赠予父亲（¥30万资产流出）**
- 鲨鱼记账：✅ 支出「长辈」¥30万，备注「X5赠予父亲」
- balance_sheet：下季末删除 X5 行（`Asset_RealEstate`）

> 金额填 balance_sheet 提计残值（¥30万），非市场价，保持口径一致。

**② X3 卖出变现 ¥20.7万**
- 鲨鱼记账：✅ 收入「其他收入」¥20.7万，备注「X3卖出（父亲补偿换车）」
- balance_sheet：无需操作

**③ 政府补贴 ¥1.5万**
- 鲨鱼记账：✅ 收入「其他收入」¥1.5万，备注「以旧换新政府补贴」
- balance_sheet：无需操作

**④ 贷款 ¥30万放款**
- 鲨鱼记账：❌ 不记
- balance_sheet：`Liability_LongTerm` 新增车贷行 ¥30万

**⑤ 首付 ¥1.9万 → 蔚来 ES8**
- 鲨鱼记账：❌ 不记（资产转换）
- balance_sheet：`Asset_RealEstate` 新增蔚来 ES8，提计 ¥30万

**⑥ 每月 ¥5,000 本金还款（免息）**
- 鲨鱼记账：✅ 支出「债务还本」¥5,000，备注「蔚来车贷本金」
- balance_sheet：每季末查 App 更新车贷剩余本金

**⑦ 每月 ¥1,068 电池租赁（BaaS）**
- 鲨鱼记账：✅ 支出「交通」¥1,068，备注「蔚来电池租赁」
- balance_sheet：无需操作

### 换车事件汇总

| 项目 | 鲨鱼记账 | balance_sheet 变化 |
|------|---------|-------------------|
| X5 赠予父亲 | ✅ 支出「长辈」¥30万 | Asset_RealEstate 删除 X5 行 |
| X3 卖出 ¥20.7万 | ✅ 收入「其他收入」 | 无 |
| 政府补贴 ¥1.5万 | ✅ 收入「其他收入」 | 无 |
| 贷款 ¥30万 | ❌ 不记 | Liability_LongTerm +¥30万 |
| 首付 ¥1.9万 | ❌ 不记（资产转换） | Asset_RealEstate 新增蔚来 ¥30万 |
| 月供本金 ¥5,000 | ✅ 支出「债务还本」 | 每季末更新贷款余额 |
| 电池租赁 ¥1,068/月 | ✅ 支出「交通」 | 无 |

## 工资与收入的记录口径

鲨鱼记账收入 = **税后实际到账金额**，不区分谁的工资，统一记入家庭账本。

| 收入类型 | 鲨鱼记账 | 备注 |
|---------|---------|------|
| 工资（税后到账） | ✅ 类别「工资」 | 含奖金，以实际到账为准 |
| SAP ESPP/RSU 现金到账 | ✅ 类别「工资」或「投资收益」 | 股票形式归属不记，已在 portfolio.csv 追踪 |
| 公积金（单位部分） | ❌ 不记 | 不经过银行账户，季末查余额更新 balance_sheet |
| 公积金（实际提取到账） | ✅ 类别「公积金」 | |
| 房租收入 | ✅ 类别「房租」 | |
| 理财/基金分红现金到账 | ✅ 类别「投资收益」 | |
| 家庭基金取出 | ❌ 不记 | 已在 portfolio.csv Cash NCF 追踪 |

## cashflow_log.csv 的定位

**只补录鲨鱼记账捕捉不到的特殊项**，大多数季度可以为空。

需要填：大额现金交易、跨境转账、家庭内部大额借款。
不需要填：工资/房租/消费（鲨鱼已记）、基金注资取出（portfolio.csv已记）、贷款本金变化（balance_sheet已记）。

## 季末对账流程（约 15-20 分钟）

1. **鲨鱼记账导出**：导出当季明细 CSV，放入 `$FAMILYFUND_DATA/<季度>/鲨鱼记账明细.csv`
2. **更新 balance_sheet.csv**：查各账户余额，更新存量快照（详见 USER_MANUAL.md 第 6 节）
3. **检查 cashflow_log.csv**：有无遗漏的基金外特殊项
4. **核对净资产公式**：
   ```
   期末净资产 ≈ 期初净资产
             + 鲨鱼收入合计
             - 鲨鱼支出合计（剔除「债务还本」分类）
             ± 资产估值变化
   ```
   误差在 ¥5,000 以内属正常。
